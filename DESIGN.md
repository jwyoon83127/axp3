# Jira Axis (jira_ax) Design Guide

안녕하세요~ 디자인실장 루이입니다! 🎨
`jira_ax` 프로젝트의 데이터 추출 도구 설계를 정리해 드릴게요. 대표님께서 데이터를 더 편하게 보실 수 있도록 감각적으로 구성했습니다!

## 📌 개요
Jira Cloud API를 통해 특정 프로젝트의 모든 이슈와 댓글 데이터를 수집하고, 분석에 용이한 CSV 형식으로 추출하는 도구입니다.

## 🛠 주요 기능
- **이슈 상세 정보 수집**: 제목, 고객사(제목 태그), 본문, 생성일 등
- **댓글 데이터 통합**: 이슈별로 작성된 모든 댓글을 추출하여 통합 CSV 생성
- **ADF (Atlassian Document Format) 변환**: Jira V3 API의 복잡한 JSON 구조를 사람이 읽기 편한 일반 텍스트로 자동 변환

## 📄 데이터 구조 (CSV Columns)
| 컬럼명 | 설명 | 비고 |
| :--- | :--- | :--- |
| **이슈키** | Jira 이슈 번호 | 예: AS-123 |
| **고객사** | 제목의 `[...]` 에서 추출한 이름 | 미분류 시 '미분류' |
| **제목** | 이슈의 요약(Summary) | 줄바꿈 제거 처리 |
| **이슈본문** | 이슈의 상세 내용(Description) | ADF 텍스트 변환 |
| **댓글작성자** | 댓글을 쓴 사람의 이름 | |
| **댓글내용** | 실제 댓글 텍스트 | ADF 텍스트 변환 |
| **생성일** | 이슈가 생성된 날짜 및 시간 | |

## 🚀 사용 기술
- **Language**: Python 3.x
- **Libraries**: `requests` (API 통신), `csv`, `re`, `json`
- **API**: Jira Cloud Search API V3

## 💡 참고 사항
- `JIRA_API_TOKEN` 환경 변수를 설정하여 권한을 관리하면 더욱 안전해요!
- 현재 수집된 데이터는 `AS` 프로젝트를 기준으로 최적화되어 있습니다.
